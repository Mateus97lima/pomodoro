Projeto de aplicação web baseado na técnica Pomodoro, desenvolvido para auxiliar na gestão do tempo e aumento da produtividade. A aplicação foi construída utilizando HTML5, Tailwind CSS, JavaScript, React.js e TypeScript, com foco em código limpo, componentização e boas práticas de desenvolvimento.

O projeto representou um desafio técnico significativo, exigindo resolução de problemas, adaptação a novos conceitos e aplicação de metodologias ágeis, contribuindo para o fortalecimento das minhas habilidades em desenvolvimento Front-End.
